# Accessibility

## Ziel
Digitale Produkte für alle Menschen nutzbar machen — inklusive Menschen mit Behinderungen.

## Standards
- WCAG 2.2 (Level AA als Mindeststandard)
- BFSG (Barrierefreiheitsstärkungsgesetz, ab 28.06.2025)
- EN 301 549

## Checkliste
- [ ] Kontrastverhältnisse ≥ 4.5:1 (normaler Text)
- [ ] Alle interaktiven Elemente per Tastatur bedienbar
- [ ] Alt-Texte für Bilder vorhanden
- [ ] Semantisches HTML
- [ ] Fokus-Indikatoren sichtbar
- [ ] Screenreader-Test durchgeführt
